关于800Bank

800Bank立志于成为中国最受推崇的互联网私人财富管家。经过多年投行业务和资产管理的耕耘， 800Bank凭借着对产业和金融的深入理解，并积极响应大众创业万众创新的新国策，一举推出互联网金融3.0模式，不再局限于P2P，而是提供投贷联动、股权众筹、跨境投资等综合性的资本市场投融资方式，实现多维度的财富管理。

800Bank隶属于仁和智本资产管理集团。 经历了2008年金融危机，一夜之间曾经令人崇敬和向往的西方金融体系的坍塌，一群有理想有勇气的中国金融人于2010年创立了仁和智本，并立志于将其打造成定位于本土的新一代投行和资产管理机构。 五年过去了， 仁和智本在业界小有名气，累计资产管理规模超过人民币120亿，并将业务拓展至曾令金融人士仰慕的美利坚大地。 优秀的投资理念和良好的企业文化也吸引了数位金融业高管，著名的金融学教授和资深的律师加盟，并汇聚了众多的企业家，一起探索产金结合提升实体经济之大道。 过去两年， 仁和智本也积极研究互联网金融，也涌现出著名的互联网金融评论家和实践者——江南愤青，位列2014年互联网50人之列。

近年来，全球经济发生巨变， 中国实体经济持续下滑、贸易失衡，而股市如脱缰的野马狂飙。 究竟是拥抱非理性的繁荣还是拭目以待潮退后的裸泳，莫衷一是。 唯一可以确定此次泡沫运动将带来剧烈的财富再分配，普通投资人将如何面对？ 在中国市场吃发展红利，去海外市场避动荡风险，现已成为许多高净值人士的理性投资选择，而普通投资者只能望而生叹。 新三板是风口上的猪，私募基金叱咤风云，面对着数百万的投资门槛，普通投资者只能隔岸观望。中国大妈们将毕生的血汗钱投入股市与专业投资人搏杀，承受着心脏的高压或是索性把钱存在银行两耳不闻窗外事，却忍受着别人赚钱时内心的煎熬。 怀着普惠的理想，我们成立800Bank，由衷希望让投资理财不再有高门槛，而投资更是掌上五分钟的滑动，尽享投资乐趣。 亲爱的投资者，请把你们宝贵的时间浪费在美好的事物上，让800Bank为您打理您的财富。

团队介绍

  • 吴立峰

    总裁

    吴立峰先生是外资金融机构早期在国内培养的优秀高级管理人和银行家，拥有近二十年的丰富管理经验。在财富管理，跨境融资及各类金融产品的优化组合和风险管理领域均具备深厚造诣。

    在创立800Bank之前，吴立峰先生历任汇丰银行青岛分行副行长，重庆分行行长，中国区工商业务总监及恒生银行上海分行行长。

    吴立峰先生毕业于复旦大学数学系，特许金融分析师(CFA)，并拥有上海高级金融学院EMBA硕士学位。

  • 周君强

    常务副总裁

    在创立800Bank之前，周君强先生曾任上海市公房资产经营（集团）有限公司副总裁、上海宝德置业有限公司总经理、内蒙古天龙集团执行总裁、锦天城律师事务所合伙人，多年来致力于产业并购重组及投融资方面的实践和创新，尤其在房地产、水务及生态环保领域有众多成功操作案例并积累了丰富的实践经验。

    周君强先生曾是中国律师协会律师会员，并拥有长江商学院及上海高级金融学院EMBA硕士学位。

  • 高政

    董事

    高政先生是仁和智本资产管理公司董事长，在创立800Bank之前，曾担任浙江香溢融通控股集团股份有限公司副总裁、首席风控官、香溢融通控股集团金融事业总部首席执行官、浙江元泰典当有限责任公司总经理等。

    高政先生长期致力于结构性融资与固定收益投资风险管理体系的研究和实践，到目前为止，已经主导了数十亿人民币成功的夹层融资、债券投资与股权投资等。

    高政先生毕业于浙江大学历史系，并拥有长江商学院及上海高级金融学院EMBA硕士学位。

  • 陈宇

    顾问委员会主席

    陈宇先生，网名“江南愤青”基层信贷工作十余年，对民间金融有深刻认识和理解。获得过银行系统浙江省优秀客户经理，全国先进个人称号。从06年迄今写作金融评论文章一百余篇，在宏观经济、金融、互联网金融、小微信贷、财富管理等领域都有深入的调研和研究。陈宇先生的在互联网金融领域的著作包括《支付革命》、《风吹江南之互联网金融》等。

    陈宇先生目前是仁和智本资产管理集团互联网金融产业合伙人，新浪财经特约撰稿人，刘鸿儒金融法律研究院特约研究员。

  • 范咏晔

    副总裁

    范咏晔女士拥有近15年金融行业从业经验，熟悉金融市场运作模式，对高净值客户的资产配置、个人财富规划、及产品风控管理等方面有着丰富的经验。在外资银行获得卓越的成绩和丰富的财富管理经验，在市场营销及客户维护等领域有独到的见解和创新思路。

    范咏晔女士拥有美国国立大学工商管理学硕士学位，在加入800Bank之前曾担任过恒生银行个人业务副总裁、新天地支行行长；荷兰银行及花旗银行个人业务销售总监等职位。

  • 吴国俊博士

    首席经济学家

    吴国俊博士是仁和智本资产管理公司高级合伙人，休斯敦大学鲍尔商学院终身教授。在加入800Bank之前，还曾担任过上海高级金融学院副院长，上海证券交易所高级金融专家，留美中国金融学会会长，具有二十多年为大型国际投行及资产管理机构提供咨询顾问经验。作为一名金融经济学家，吴国俊博士从将从国际宏观经济形势为800Bank带来全球视野的投资理念及战略指导。

    吴国俊博士毕业于斯坦福大学，并拥有金融学博士学位。

  • 严宏超

    产品总监

    在加入800Bank之前，严宏超先生曾就职于光大证券资产管理部、上海和盛天成投资有限公司等。有着近十年的商业与办公房地产开发管理经历，丰富的投融资项目开发和金融产品设计经验，先后主导并参与了昆山仕泰隆股权投资基金（一期）、云南祥云飞龙股权投资基金、上海铂尔曼酒店投资基金、上海雍景台股权投资基金等项目，对投资架构的设计和风险有良好的判断和把控。

    严宏超先生毕业于复旦大学-麻省理工国际MBA和西班牙IE商学院Exchange-MBA。
