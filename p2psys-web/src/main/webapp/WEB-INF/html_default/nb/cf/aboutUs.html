致你们：

800众服没有华丽的辞藻堆砌出一篇非常正统的关于我们的文字，即使写出了非常官方的文字也无助于让正在读这封信的你们真实的了解800众服。

800众服初创者之一曾经也在北京和朋友们有过一次创业经历，痛苦并欢喜地享受过从零到一的创业旅程。在北京中关村大街奋斗的日子里，早上吃着西少爷肉夹馍，中午扫码送红包订一份外卖，偶尔听听孵化器路演大厅的路演项目，与战友们一起奋斗，相伴到黎明。后来中关村大街越来越出名，天使汇10米长度的屏幕开始每天滚动播放着各种天马行空的项目，车库咖啡、3W咖啡、京东奶茶、时光雕刻咖啡不间断地迎接着一波又一波激情四射的创业者们，甚至自己驻扎的孵化器也成为了游客参观的景点之一……后来，离开了北京，告别了战友们，回到了上海。感谢这样的一段在北京的创业经历，会让人不禁去重新反思，我们为什么而创业？我们适不适合去创业？有了很好的idea之后，该怎样让这些想法落地？……于是，有了你们现在正在浏览的800众服。

浙商创投

浙江浙商创业投资管理有限公司成立于2007年11月，中国最活跃的浙商资本，融合国际化与本土化的管理模式，短短三年，成就了中国股权投资界的“浙商经验”，目前已经成为浙江省管理资金规模最大、投资项目最多的专业股权投资管理机构之一。目前已经管理了杭州钱江浙商现代服务业基金、浙商诺海低碳基金等9个VC/PE基金和一个天使基金，可投资资金规模30多亿元人民币，管理资产规模达80亿元人民币。

上海联创

上海联创投资管理有限公司成立于1999年7月，股东为国家计委、国家经贸委、中国科学院科技促进经济基金委员会和上海联合投资有限公司。投资领域包括信息技术、生物医药、环保与新材料等。

上海城创

上海城创投资管理股份有限公司成立于2014年，是一家涵盖私募基金、开发管理、产业服务、私募股权投资等四大业务范围的综合性管理公司，专注于投资上海新一轮城市更新和产业升级过程中的城市产业综合体这一新兴业态。

同道资本

同道资本是一家专业从事私募股权基金的投资管理公司，投资范围涵盖一、二级市场的PRE-IPO投资银行业务、PE投资、证券投资、置业投资、夹层基金投资等金融领域。

浙江股权交易中心

浙江股权交易中心（以下简称“中心”）成立于2012 年10 月18 日，是浙江省政府根据中国证监会的有关文件精神，结合浙江省经济金融特色，为着力解决“两多两难”，推进温州金融综合改革，促进经济转型升级而设立的省级股权交易平台，是我国多层次资本市场体系的重要组成部分。
