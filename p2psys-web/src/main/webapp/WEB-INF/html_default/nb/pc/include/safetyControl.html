双重风险保障金制度

产品均来自于各大金融机构或类金融机构以及知名的资产管理公司，所有入选产品均提供信用增级与安全保障措施。此外，平台的金融产品团队来自于知名外资银行、四大会计事务所以及资产管理公司，建立起了“事前预防、事中监督、事后应对 ”的安全风险体系。事前确保每个上线项目均需通过三重风险审核把控，事中确保合作机构实施合作跟踪管理，事后确保对合作机构重新进行风险评级。

第三方支付机构银联支付与连连支付强强联手提供双支付通道，全维度辅助平台实施资金监管和交易监测。同时，平台联手国有五大商业银行之一的建设银行 独立开设投资本金账户和收益分配账户，独立分账管理每个产品，所有资金走向均受清晰地监管，为投资人的资金安全保驾护航。

1%的风险保障金制度与风险准备金保障制度提供双重保障。对于所有在平台线上发布的产品，平台按每笔成交额的1%计提风险保障金独立账户存放。此外，当1%的风险保障金不足以垫付出现的投资损失时，平台可用逾千万元的风险准备金垫付投资本金及收益。一旦出现产品逾期，平台将通过风险保障金和风险准备金两种途径垫付，并将主动协助投资人通过法律途径解决违约问题。

IT团队来自于国内各大银行，在信息安全和数据安全方面有着非常丰富的经验，让投资人享受银行级别的安全保障；基于诺顿的SSL证书（https）安全传输，三层防火墙隔离系统的访问层、应用层和数据层集群；专业黑客流量攻击入侵防范及阿里云完备的容灾备份，确保交易数据安全无虞。

母公司仁和智本是中国新一代投资银行及资产管理机构，总部位于中国上海和杭州，并在美国纽约、休斯顿等地设有分支机构。截止2014年底，集团累计资产管理规模超120亿元，旗下管理的产品100%到期按时兑付，为投资人取得优异投资收益。
