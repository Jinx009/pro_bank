首先感谢大家对800Bank平台的信任与支持！

800Bank新手注册即享8000元体验金活动于9月29日零点正式截止。由于近期我们发现有一批恶意刷取体验金的用户。为了维护平台的交易环境和投资人公平享受活动的权利，根据平台运营数据监测以及具体的注册渠道、服务器日志、用户登录、投资行为等多方面判断，我们对9月27日起注册的用户特作出如下决定，在保护用户身份信息和账户安全的前提下：

1、对于使用体验金前未充值认证的客户取消体验金活动资格，所获得的体验金收益属 于非常规获取，不予以发放。

2、对充值未进行任何投资的用户取消体验金活动资格，所获得的体验金收益不予以发放。

3、对已经投资的用户，若可以核对身份信息，则正常通过提现审核（本金+投资收益+体验金收益）；若无法核对身份信息，可通过本金+投资收益的提现审核。

出于对投资安全及用户信息的保护、对违法行为的抵抗，800Bank将继续对可疑账户进行验证及调查，同时封禁可疑账户并保留将此违法行为向公安机关举报的权利。为此给用户带来的不便敬请谅解。
